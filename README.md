# Smart Tourism System

A full-stack tourism web application designed to help users explore destinations, discover accommodations, and plan memorable travel experiences.

## 🚀 Live Demo

https://smart-tourism-system.onrender.com

## 📖 About the Project

Smart Tourism System is a modern web application that provides users with an easy way to browse travel destinations, view listings, and explore tourism-related information through an interactive and user-friendly interface.

The project focuses on delivering a seamless experience for travelers while demonstrating full-stack web development concepts using Node.js, Express.js, MongoDB, and EJS.

## ✨ Features

* User Authentication and Authorization
* Browse Tourism Destinations
* Add and Manage Listings
* Image Upload Support
* Responsive User Interface
* Secure Backend Integration
* Database Management with MongoDB
* Dynamic Content Rendering using EJS

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/avinashsingh88161-sudo/wanderlast.git
```

2. Navigate to the project folder

```bash
cd wanderlast
```

3. Install dependencies

```bash
npm install
```

4. Create a .env file and add the required environment variables.

5. Start the application

```bash
npm start
```
